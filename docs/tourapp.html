Overview

The main pages provides access to the other Activities and Settings, as well as allowing easy start tracking your movement. If a GPS listener is active (due to tracking or scheduling), additional information will be displayed.
The GPS update intervals can be changed in the settings.
Upon stopping tracking, a save page will be opened, where you can enter additional information. The track will then be stored as a gpx filed in the Download folder.

map

Map

This displays an OpenTopoMap, allowing you to navigate. The tiles will be cached, providing some sort of offline availability. You can enable a latitude/longitude grid, as well as your current position. You can also show/set your home location (will be set if only one marker is on the map) Additionally you can enable waypoints, which allows you to add/remove waypoints upon presses. You can use the Graphhopper Routing Engine to find a suitable path over the waypoints. The App will tell you an approximate length, ascend and travel time. (In the current version, it uses the default Graphhopper settings 4km/h and 400hm/h, however, Elevation data is NOT super trustworthy!). Additionally a tour schedule for the waypoints can be exported, which will use the routing Engine to estimate arrival times for each waypoint. The routing requires some pre computed files for Graphhopper, which can be downloaded from inside the App (Under RoutingEngine)

Tour Schedule

In the Tour Schedule View you can open a pre-planned tour schedule (*.schedule). The Waypoints will be displayed with their GPS coordinates and their last reaching time. If the GPS is enabled, the app will regularly poll your position and will advance to the next waypoint if the previous one has been reached. The app will notify you with an alarm if you have not reached the waypoint on time. In the current Version, it will skip all waypoints which lie in the past upon loading the file.
Each line contains
longitude latitude time index name
The index is used to find the next waypoint upon turning around. (the scheduler will jump forward to the next entry with the same index)

RoutingEngine

You can change the routing area and download new areas. The top right icon indicates, whether the area is stored on the device, active in the engine or not existent on the device.

Settings

You can log your movement to a gpx file, the update intervals can be adapted in the settings.
